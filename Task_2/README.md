##  To-Do List App

A simple To-Do List web app built with **HTML, CSS, and JavaScript**.  
You can **add tasks, mark them complete, and delete them** with a clean and responsive UI.  

##  Features
- Add new tasks  
- Mark tasks as complete  
- Delete tasks instantly  

##  Tech Used
- HTML  
- CSS  
- JavaScript (ES6)
